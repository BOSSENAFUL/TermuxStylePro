### 📖 এই টুলটি আসলে কী? (What is this?)
**TermuxStylePro** হলো একটি পাওয়ারফুল টারমাক্স কাস্টমাইজেশন টুল। এটি আপনার সাধারণ টারমাক্স টার্মিনালকে একজন প্রফেশনাল হ্যাকারের মতো লুক দেয়। এই টুলটি ব্যবহার করে আপনি টারমাক্সের ব্যানার, থিম এবং ফন্ট নিমিষেই পরিবর্তন করতে পারবেন।

### 🛠️ এটি কীভাবে কাজ করে? (How it Works?)
১. **ব্যানার সেটআপ:** আপনি যখন এটি রান করবেন, এটি টারমাক্সের `.bashrc` ফাইলে আপনার নাম (BOSS ENAFUL) এবং পছন্দের লোগো সেট করে দেয়।
২. **অটো-গ্রিটিং:** টারমাক্স ওপেন করার সাথে সাথেই এটি আপনাকে ভয়েস মেসেজ বা টেক্সট দিয়ে উইশ করবে।
৩. **কালারফুল ইন্টারফেস:** এটি `lolcat` এবং `figlet` ব্যবহার করে আপনার টার্মিনালকে কালারফুল এবং স্টাইলিশ করে তোলে।
